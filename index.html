🚀 Internship Selection Prediction Project

This project uses machine learning to predict whether a student will be selected for an internship based on academic, technical, and personal factors.

📊 Project Highlights

  • Exploratory Data Analysis (EDA)
  • Data Cleaning & Preprocessing
  • Machine Learning Models
  • Model Evaluation

🔗 Run the Project

👉 Open Google Colab Notebook

📁 Dataset

Dataset included in repository.

📈 Key Insight

Interview scores and coding skills are the strongest predictors of internship selection.
